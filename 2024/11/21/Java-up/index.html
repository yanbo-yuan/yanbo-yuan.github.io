Java 的内部类

1.什么是内部类

定义：在一个类的里面，再定义一个类。

举例：在A类的内部定义B类，B类就被称为内部类

什么时候用到内部类？

2.内部类的分类

  • 成员内部类

  • 静态内部类(也是成员内部类的一种)

静态内部类只能访问外部类中的静态变量和静态方法

如果想访问非静态的需要创建外部类的对象。

总结：

  • 局部内部类(没什么用~~~可以略过)

将内部类定义在方法里面就叫做局部内部类，类似于方法里面的局部变量。

  • 匿名内部类

匿名内部类的本质：隐藏了名字的内部类。

看看这个你就能理解匿名内部类的实现方法了
